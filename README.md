# Посмотри в окно

Адаптивная страница с карточками, сеткой, прелоадерами и продуманной типографикой.

## Возможности

- Сетка карточек на Flex/Grid с переносом на новую строку.
- Прелоадеры для `.result__video-container` и `.content__list-container`.
- Кликабельные карточки (заголовок + описание).
- Обрезка текста описания через `-webkit-line-clamp`.
- Проработанная типографика: `line-height`, отступы, межбуквенные интервалы.
- Адаптивность под мобильные устройства.
